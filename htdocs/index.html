Introduction Improving your hydraulic analysis.

  ArcView 8.1 Survey (August 17th, 2001)

  

A survey whether there's need for a Version of the DC Water Design Extension for ArcView 8.1 is available here. Please take the time to fill in the survey.

  Version 2.02 released (June 24th, 2001)

  

Version 2.02 is available now (DC Water Design Extension Installer, Source Project. Changes include bug fixes, useability enhancements, an updated installer (it creates now start menu entries for the manual and the uninstaller), a configurable snapping radius and updated sample data.

  Version 2.01 released (May 31th, 2001)

  

Version 2.01 is available now (DC Water Design Extension Installer, Source Project, PDF Manual). Changes include documentation updates, support for backdrop maps (thanks to Roland Salgado), a new function to create zones from SOV controls and a bugfix for coordinate systems using decimal feet.

  Added Sample Data (April 19th, 2001)

  

Sample Shapefiles that work with the DC Water Design Extension can be are available now (example_network.zip).

  Added EPANET2MYSQL and Calibration code (March 30th, 2001)

  

EPANET2MYSQL and some calibration code that uses Genetic Algorithms can now be found in the CVS.

  DC Water Design Extension added to Sourceforge (February 28th, 2001)

  

DORSCH Consult releases the DC Water Design Extension under the GNU Lesser General Public License. An ArcView Extension that integrates the EPANET hydraulic analysis software with ArcView GIS.
